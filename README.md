# 🐟 Bite the Bait
A 2D fish survival game — 1st Semester C++ & SFML Project

## 👥 Made by
- Minahil Basalat
- Aliza Nadeem

## 🎮 How to Play
- Eat green fish to grow and earn points
- Avoid red predators or lose a life
- Collect gold stars for extra lives

## ⚙️ How to Compile & Run
g++ main.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system -std=c++17

Place arial.ttf in the same folder as the executable before running.
